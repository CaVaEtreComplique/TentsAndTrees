    class ChronoUi

    Presentation¶ ↑

    The ChronoUI class is inherited from the Button class and manages the in-game chronometer.

    Methods¶ ↑

    This class knows three methods : initialize, updateLabel (both inherited from the Button class) and the parce method. These methods are described further down.

    Variable¶ ↑

    The only variable this class needs is the time value.

    parce(time) click to toggle source

    The parce method is needed for the initialization and for the label update. It gives the time a value that is easily read by humans. By doing so, it can be displayed in the game.

    # File Game/Ui/Buttons/ChronoUi.rb, line 46
def parce(time)
  s, m = time % 60, time / 60
  h = m / 60
  m = m % 60
        [h, m, s]
end
