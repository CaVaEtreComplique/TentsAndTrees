    class ProcessStatus

    Presentation¶ ↑

    The ProcessStatus class is the representation of the design pattern observator. It gives the possibility to send messages to multiple other classes.

    Variables¶ ↑

    • status - This class variable is a String that represents the message received.

    • observers - This class variable represents the GridGenerator class.

    Methods¶ ↑

    This class knows an initialization method and the methods add and send described below.

def ProcessStatus.send(message)
  if @@status!=message
    @@status=message
    @@observers.each{ |o|
      o.recive(@@status)
      # sleep(0.5)
    }
  end
  self
end
