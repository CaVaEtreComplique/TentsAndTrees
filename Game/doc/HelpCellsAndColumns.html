    class HelpCellsAndColumns

    Presentation¶ ↑

    The HelpCellsAndColumns class is an abstract class inherited from the abstract class HelpCellsAndRowsOrColumns. This abstract class will determine a model for every help class that needs to know a cell and a column.

    This class' direct child is the HelpAllPossibilitiesGiveItColumn class.
