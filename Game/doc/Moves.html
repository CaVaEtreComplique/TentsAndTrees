    class Moves

    Presentation¶ ↑

    This class manages all the moves within the game, epecially the undo/redo function.

    Methods¶ ↑

    This class' methods are undo, redo, replay, add and clear and are applied to the moves, using a redo stack.

    Variables¶ ↑

    • moves - All the moves.

    • redo - Redo stack.

    # File Core/Moves.rb, line 90
def redo
        if @redo.last != nil
                @moves.push(@redo.pop)
                return @moves.last
        end
        return [[[0,0]],"outOfBound"]
end
